## General info
This project is a web application.  
The main purpose is to create a currency converter app.  

## Technologies
Project is created with:
* Java 11 
* Spring Boot 2.4.1
* Spring Cloud 2020.0.0  
* Spring Data JPA  
* Maven  
  
IDE: Intellij IDEA 2020.2
